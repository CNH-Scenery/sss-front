# Tacit Trader

암묵지(暗默知) 기반 매매 전략 코드화 워크플로우. Claude Design 프로젝트
`Tacit Trader.dc.html`를 Vite + React 앱으로 구현한 것입니다.

## 워크플로우 (4단계)

1. **설문** — 미래를 가린 과거 차트 10개에 대해 매수/매도/관망을 자연어 근거와 함께 응답
2. **전략 코드화** — 누적된 응답 + 지표 스냅샷에서 임계값을 역설계해 파이썬 `decide()` 함수 생성 (정제·버전 이력 포함)
3. **백테스트 & 자기일관성** — 전략을 과거 구간에 적용하고 설문 시점의 실제 선택과 비교
4. **실시간 모니터링 & 알림** — 현재가 폴링 → 전략 신호 계산 → 알림 피드 (주문은 사용자가 직접)

> 읽기 전용 시세만 사용하며 주문은 실행하지 않습니다. 분석·알림까지가 앱의 역할입니다.

## 실행

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5173)
npm run build    # 프로덕션 빌드
npm run preview  # 빌드 결과 미리보기
```

## 정밀 인식 결제

- 기본 설문 5개를 완료하면 전략 코드 생성이 가능합니다.
- 6~10번째 추가 설문은 정밀 인식 패스(1회 ₩9,900) 결제 후 저장됩니다.
- 실제 결제 페이지가 있으면 `.env`에 `VITE_PREMIUM_PAYMENT_URL=https://...`를 설정하세요.
- 결제 링크가 없을 때는 해커톤 데모용으로 버튼 클릭 시 즉시 정밀 인식 패스가 활성화됩니다.

## 구조

- `src/App.jsx` — 메인 컴포넌트. 차트/지표/백테스트/코드 생성 로직과 화면 렌더링
- `src/css.js` — 인라인 CSS 문자열을 React style 객체로 변환하는 헬퍼
- `src/index.css` — 전역 스타일 (스크롤바, 애니메이션)
- `index.html` — Pretendard / JetBrains Mono 폰트 로드

모든 시세·지표 데이터는 시드 기반으로 결정론적으로 생성되는 데모 데이터입니다.
